## What is Re:MirAI?

Re:MirAI is an AI-powered personality discovery platform that answers the question: **"How do others see me?"**

By collecting anonymous feedback from friends through shareable surveys (called "Rituals"), Re:MirAI synthesizes a unique AI persona that reflects your external personalityâ€”creating an interactive chatbot you can engage with to explore the gap between self-perception and how others perceive you.

## Key Differentiators

**Privacy-First Approach**
- Anonymous feedback collection
- User control over persona visibility
- No data selling or third-party sharing

**AI-Powered Synthesis**
- Advanced personality profiling using GPT-4
- Multi-dimensional analysis
- Contextual chat responses

**Social & Engaging**
- Shareable persona cards
- Compatibility matching
- Gamification elements
