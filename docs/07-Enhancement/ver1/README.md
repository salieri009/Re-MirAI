## Design Systems

### Color Palette
- **Small Switch Palette**: See [09-Color-Palette-Plan.md](./09-Color-Palette-Plan.md)
- **Muted Colors**: Updated to less bright tones for better UX

### Spacing System
- **4px Baseline Grid**: See [10-4px-Baseline-Grid-System.md](./10-4px-Baseline-Grid-System.md)
- All spacing, dimensions, and offsets must be multiples of 4px

### Typography System
- **Three Font Families**: See [11-Typography-System.md](./11-Typography-System.md)
  - **Poppins** (Display): Brand, headlines, magic
  - **Inter** (Body): Content, readability
  - **Space Mono** (Tech): Data, stats, echoes count - "AI technology behind the magic"

### Compliance
- **KickoffLabs Requirements**: Landing page follows all 12 requirements
- **WCAG 2.1 AA**: All pages maintain accessibility standards
- **Nielsen's Heuristics**: UX principles applied throughout

## Quick Reference

### Spacing Tokens (4px Grid)
- `--space-xs`: 4px
- `--space-sm`: 8px
- `--space-md`: 16px
- `--space-lg`: 24px
- `--space-xl`: 32px
- `--space-2xl`: 48px
- `--space-3xl`: 64px

### Border Radius (4px Grid)
- `--radius-sm`: 4px
- `--radius-md`: 8px
- `--radius-lg`: 12px
- `--radius-xl`: 16px

### Typography
- Font sizes: 12px, 14px, 16px, 18px, 20px, 24px, **32px** (was 30px), 36px, 48px
- Line heights: Adjusted to create 4px-aligned baselines
